# Gestionnaire_de_fichiers_chiffres

Application de gestion de fichiers chiffrés développée dans le cadre du cours de Sécurité des applications.
L’application permet de chiffrer et déchiffrer des fichiers en utilisant AES-256 en mode CBC et d’assurer leur intégrité grâce à un HMAC-SHA256.

Les opérations cryptographiques sont exécutées dans un processus séparé afin d’améliorer la sécurité et la robustesse de l’application.
